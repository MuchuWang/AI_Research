# Generative AI Research Project

## Project Overview

This repository organises research materials for the project titled **Using Generative AI Tools: Boon or Bane**. It includes literature review materials, quantitative and qualitative research data, draft reports, and additional project resources.

## Folder Structure

### 01_literature_review
This folder stores literature review materials, including journal articles, conference papers, books, and news articles.

### 02_quantitative_analysis
This folder stores survey data, survey questions, cleaned datasets, analysis scripts, and survey analysis reports.

### 03_qualitative_analysis
This folder stores interview transcripts, interview protocols, consent forms, thematic analysis files, and qualitative insights.

### 04_drafts_and_reports
This folder stores draft research proposals, conference papers, and final reports.

### 05_additional_materials
This folder stores information sheets, photos, media files, and other supporting materials.

## Naming Convention

Files should use clear, consistent, and descriptive names. A recommended naming format is:

`YYYYMMDD_description_version_author`

Example:

`20260428_survey_data_cleaned_v01_MW.csv`

A good naming convention helps researchers identify files quickly, track different versions, and reduce confusion when multiple people work on the same project.

## Data Access and Privacy

Sensitive data, such as interview transcripts and consent forms, should be stored securely with restricted access. Personal or identifiable information should not be placed in a public repository.

## Contribution Guidelines

Future collaborators should:

1. Use clear and consistent file names.
2. Place files in the correct folder.
3. Write meaningful commit messages.
4. Avoid overwriting other collaborators' work.
5. Document major changes in the README or project log.
